## 🚀 快速开始

### 1. 小红书Agent系统
```bash
cd src/xiaohongshu_agent
pip install -r requirements.txt
python main.py
```

### 2. 在线应用
```bash
cd ai-coffee-salon-web
npm install
npm run dev
```

### 3. 知识涌现系统
```bash
cd src/knowledge_emergence
pip install -r requirements.txt
python main.py
```
