## 技术实现

### 前端技术栈
- **框架**: React 18 + TypeScript
- **样式**: Tailwind CSS（自定义设计系统）
- **图标**: Lucide React（严格遵循无emoji规范）
- **动画**: Framer Motion
- **图表**: ECharts + ECharts-for-React
- **构建**: Vite 6

### 设计系统
- **主色调**: 深绿咖啡色系（#1a3a2e - #5fb99d）
- **强调色**: 金色系（#7a5a1a - #f5c563）
- **辅助色**: 深棕咖啡色系（#2b1d14 - #a97655）
- **字体**: 系统字体栈（优先PingFang SC, Microsoft YaHei）
- **圆角**: 渐进式圆角系统（sm 4px ~ 2xl 32px）
- **阴影**: 多层级阴影系统（含特殊发光效果）

### 性能优化
- ✅ 代码分割
- ✅ 组件懒加载
- ✅ 动画性能优化（使用transform和opacity）
- ✅ 生产构建压缩（gzip: 101.91 kB）

## 设计特色

### 视觉设计
1. **咖啡主题色彩**: 深绿、金色、棕色的和谐搭配，体现咖啡文化的温暖与品味
2. **渐变背景**: Hero区使用品牌色渐变，营造科技感和专业感
3. **毛玻璃效果**: 部分卡片使用backdrop-blur，增强层次感
4. **发光效果**: CTA按钮和重点元素使用shadow-glow，突出视觉焦点

### 交互设计
1. **平滑滚动**: 所有锚点导航使用smooth scroll
2. **动画时序**: 使用staggerChildren实现渐进式动画
3. **Hover反馈**: 所有可交互元素提供明确的hover状态
4. **Tab切换**: 协议区使用AnimatePresence实现流畅切换

### 响应式设计
1. **移动端优化**: 汉堡菜单、单列布局、字体缩放
2. **平板适配**: 2列网格布局
3. **桌面端**: 完整3-4列布局
